## Requirements

- Node.js (v16.13.2)

- npm or yarn

## SetUp

First, install dependencies:

```console
$ npm install

or

$ yarn
```

Then, start [our api](https://github.com/crud-client/api) in the background

## Usage

To run this app, use:

```console
$ yarn start
```
